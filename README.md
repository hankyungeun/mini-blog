# 미니 블로그
#### 인프런강의 - 처음만난 리액트(미니블로그 만들기 클론코딩)

### 기능
- 글 목록 보기 기능
  PostList, PostListItem
- 글 보기 기능
  Post
- 댓글 보기 기능
  CommentList, CommentListItem
- 글 작성 기능
  PostWrite
- 댓글 작성 기능
  CommentWrite

### 폴더 구성
- src/component/list : 리스트와 관련된 컴포넌트
- src/component/page : 페이지 컴포넌트
- src/component/ui : UI컴포넌트
